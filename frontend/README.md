# AI-Powered Learning Assistant

Live Demo: https://your-frontend-url.vercel.app/signUp
(Sign Up is required for a demo)

## Features
- Document upload
- AI Q&A (RAG)
- Firebase Authentication
- Vector search

## Tech Stack
Frontend: React, Tailwind  
Backend: Node.js, Express  
Vector DB: Qdrant  
AI: OpenAI / Gemini

## Setup
1. Clone repo
2. Add .env
3. npm install
4. npm run dev
